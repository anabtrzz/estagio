Notas musicais

As notas são a menor parte do que é classificado como som. O jeito mais simples de pensar para se entender a definição de uma nota é imaginar um acorde. Acordes são um conjunto de notas – como blocos alinhados, que quando são desmontados, o que se tem são as notas separadas.


Notas são formadas por frequências, que quando alteram o movimento das moléculas no ar, formam o som. A notação é representada de uma forma universal, utilizando uma sequência de códigos para facilitar o entendimento de músicos que falam linguagens diferentes.


A sequência de notas é

DÓ – RÉ – MI – FÁ – SOL – LÁ – SI

Que na linguagem universal são representadas, respectivamente, por:

C – D – E – F – G – A – B

Recapitulando:
Dó – C
Ré – D
Mi – E
Fá – F
Sol – G
Lá – A
Si – B
As cifras são lidas e escritas apenas através da linguagem universal, citada acima.

Notação

As cifras e tablaturas são lidas e escritas apenas através da linguagem universal

Dó – C
Ré – D
Mi – E
Fá – F
Sol – G
Lá – A
Si – B


  • Tablatura

    • A contagem das cordas do violão se dá de baixo para cima, sendo a nota mais aguda, a primeira corda, e a mais grave, a última. A ordem das notas no violão fica assim:

    1º Mi – E (aguda)
    2º Si – B
    3º Sol – G
    4º Ré – D
    5º Lá – A
    6º Mi – E (grave)


    Sabendo disso, temos a escrita da tablatura da mesma forma, usando as letras da linguagem universal na ordem de cada corda do violão, assim:

    E|
    B|
    G|
    D|
    A|
    E|

    Logo ao lado das letras, após a barra, é onde é mostrada a posição dos dedos em cada corda. Agora observe o braço do violão, do início do braço até a caixa acústica, existem 12 casas, considere que cada casa seja numerada de acordo com a sua posição, do 1 até o 12. Essa numeração aparecerá, como já foi dito, ao lado das letras que equivalem a cada corda, sendo assim a informação que a pessoa que está lendo a tab terá, vai ser da corda que vai ser tocada e da casa que o dedo da pessoa pressionará a corda.

    Dessa forma:
    E|-1---------------
    B|-----------------
    G|-----------------
    D|-----------------
    A|-----------------
    E|-----------------

    Nesse exemplo, a corda que a tab pede que seja tocada é a primeira corda, na primeira casa, soando a nota Fá.
    A dúvida de saber qual nota se está tocando geralmente surge nesse momento. Novamente, o violão trabalha com sequências, cada corda se inicia de uma nota, que equivale à corda própria solta, já que estamos utilizando o exemplo da primeira corda, a Mi aguda, vamos continuar por ela.

    Corda Mi:
    ----|----|----|----|----|----|----|----|----|----|----|----

    Cada tracinho acima equivale a sua posição no braço do violão, se a corda solta é Mi, quando pressionada na primeira casa, é Fá, porque depois do Mi na sequência DÓ – RÉ – MI – FÁ – SOL – LÁ – SI, depois do Mi vem o Fá.

    Agora, voltando para o que já foi ensinado sobre tons e semitons:
    Sabemos que um tom equivale a dois semitons juntos, e sabemos também que após o Mi vem o Fá, pois existe apenas um semitom entre os dois pelo fato de não existir um sustenido de Mi, então após o Mi é, automaticamente um Fá. Mas entre o Fá e o Sol temos o Fá sustenido, que juntando tudo dá um tom, pois a sequência fica:

    FÁ – FÁ# – SOL
    Dois semitons,
    ou seja,
    Um tom.

    Então a segunda casa pressionada soa Fá Sustenido, seguindo a mesma lógica, a terceira casa soa Sol, e a quarta Sol Sustenido. E segue a mesma ordem até o Si, porque também já se sabe que do Si para o Dó acontece o mesmo do que do Mi para o Fá.
    Si não tem sustenido.
    Mi não tem sustenido.

    O sustenido de cada um deles é a próxima nota, nesse caso, após o Si, como não existe um Si sustenido, a próxima nota será o Dó.

  • Cifras

    • Ler uma cifra é muito parecido com ler uma tablatura, a primeira diferença é que enquanto a tablatura mostra apenas como tocar uma nota de cada vez, a cifra mostra como tocar um acorde, ou seja, mais de uma nota ao mesmo tempo.
    A leitura da cifra é vertical, assim:

    | | | | | |
    | | | | | |
    | | | | | |
    | | | | | |
    | | | | | |
    | | | | | |
    | | | | | |

    (Imagine o braço do violão com as seis cordas viradas para você);
    Exemplo: Acorde de Lá Maior

    A
    | | | | | |
    | 1 2 3 | |
    | | | | | |
    | | | | | |
    | | | | | |
    | | | | | |
    | | | | | |

    Outra diferença é que para ler uma cifra, a numeração, ao invés de dizer em qual casa se deve pressionar, significa qual dedo usar para pressionar a corda. Imagine que o seu dedo indicador é o dedo 1, e todos após ele são respectivamente 2, 3 e 4. No exemplo acima, em Lá Maior, o dedo 1 aperta a 5ª corda, enquanto o dedo 2 aperta a 4ª corda, e o dedo 3, por sua vez aperta a 3ª corda.
